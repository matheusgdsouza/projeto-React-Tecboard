## 📌 Sobre o Projeto

O **TecBoard** é uma aplicação front-end construída para simular uma plataforma de publicação de conteúdos, onde usuários podem visualizar e criar posts.

O projeto foi desenvolvido com o objetivo de consolidar conhecimentos em React e boas práticas do ecossistema moderno de front-end, incluindo componentização, gerenciamento de estado e consumo de dados.

---

## ⚙️ Conceitos e Métodos Aplicados

### 🔸 Componentização

A aplicação foi estruturada em componentes reutilizáveis, facilitando manutenção e escalabilidade.

### 🔸 Hooks do React

Uso de hooks como:

* useState → controle de estado local
* useEffect → efeitos colaterais e ciclo de vida

### 🔸 Renderização Dinâmica

* Listagem de conteúdos baseada em dados
* Atualização automática da interface conforme o estado muda

### 🔸 Props (Propriedades)

* Comunicação entre componentes pai e filho
* Reutilização de lógica e estrutura

### 🔸 Manipulação de Eventos

* Interações do usuário (cliques, inputs, etc.)

### 🔸 Organização de Pastas

* Separação clara entre componentes, páginas e estilos
* Estrutura pensada para crescimento do projeto
